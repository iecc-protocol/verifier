# IECC Network & Consortium

The IECC Network (Independent Credential Infrastructure) and the IECC Consortium are built on the principle that professional trust should be open, verifiable, and permanent. We welcome developers, educators, and organizations to join our ecosystem.

## How to Contribute

We are looking for help in the following areas:

1.  **Language SDKs**: Porting the IECC Verifier to Python, Go, Rust, and Java.
2.  **Integrations**: Developing plugins for LMS platforms like Moodle, Canvas, or WordPress.
3.  **Documentation**: Improving technical standards and implementation guides.
4.  **Security Audits**: Reviewing our cryptographic implementations.

## Backlink & Recognition Program

If you are using IECC to issue or verify credentials, we encourage you to:

-   **Link to the Verification Hub**: Use `verify.iecc.world` in your certificate footers.
-   **Showcase on GitHub**: Add the "Verified by IECC" badge to your organization's README.
-   **Developer Registry**: Contributors with significant PRs will be issued an **Open Source Contributor Credential** (verifyable via this tool).

## Verified Issuers

To be added to the [Trusted Issuers Registry](./trusted-issuers.json), please submit a Pull Request following the established schema.
